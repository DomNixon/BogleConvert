## üìä Cloudflare Web Analytics

We use **Cloudflare Web Analytics** to monitor site performance and health.

### What this means for you:

- **No cookies.** Cloudflare Web Analytics does not use cookies or any form of client-side storage.
- **No cross-site tracking.** It does not track individual visitors across the web.
- **No personal data collection.** It does not collect IP addresses, user agents beyond aggregated device types, or any personally identifiable information.

### What we see:

- Aggregated page views and visit counts
- General geographic regions (country-level)
- Performance metrics (page load times, error rates)
- Device type breakdowns (desktop vs. mobile)

This data is used solely to understand overall site traffic patterns and ensure the application is performing well for all users.

For more information, see [Cloudflare's Web Analytics privacy documentation](https://www.cloudflare.com/web-analytics/).
